# SpringCloud-Netflix 技术点

1. Eureka：服务注册与发现，用于服务管理。

2. Feign： web调用客户端，能够简化HTTP接口的调用。

3. Ribbon：基于客户端的负载均衡。

4. Hystrix：熔断降级，防止服务雪崩。

5. Zuul：网关路由，提供路由转发、请求过滤、限流降级等功能。

6. Config：配置中心，分布式配置管理。

7. Sleuth：服务链路追踪

8. Admin：健康管理
